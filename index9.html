Descripción 💯✅:

Crear una plataforma de para aprende vocabulario en ingles con las siguientes caracteristicas:

  1. Contenido de 2000 palabras
  2. Seccion para usuario para que personalice las palabras aprendidas
  3. Seccion para administrador para que obtenga la cantidad de usuarios y cantidad de palabras aprendidas
  4. Seccion para vocabtest de las palabras
  5. Seccion para gramatica
  6. Seccion para que aparezcan todas las palabras se puedan identificar como facil y dificil para asi poder crear una segregacion de forma que se repase las mas dificiles o faciles
  7. No es necesario que todas las palabras tengan audio y sus caracterista como: imagen, audio, palabra ingles, palabra español y traduccion fonetica, solo que contenga algunas
  8. Hacer que la plataforma llegue a 20 clientes activos al mes
  9. Crear una comunidad a partir de ello
  10. Integrarle IA para conversaciones, club de conversacion, pronunciacion, gramatica, flashcard, videos, frases mas conocidas para que el estudiante al usarla aprenda si o si de manera mas rapida el ingles
